# 🧠 Чистый Интеллект

Локальный оркестратор для LLM с иерархической памятью и управлением контекстом.

## Что это?

**Чистый Интеллект** — это промежуточный слой между пользователем и локальной LLM (Ollama), который:
- Индексирует код проекта через tree-sitter
- Хранит знания в векторной БД (ChromaDB) и графе связей (NetworkX)
- Динамически собирает оптимальный контекст для каждого запроса
- Работает как **stateless proxy** — модель не хранит историю, Оркестратор управляет памятью

## Архитектура

```
┌──────────────┐     ┌───────────────────┐     ┌──────────┐
│ Agent Zero / │ ──→ │   ОРКЕСТРАТОР     │ ──→ │  Ollama  │
│    CLI       │ ←── │  (FastAPI :8085)  │ ←── │ :11434   │
└──────────────┘     └───────────────────┘     └──────────┘
```

### Ключевые компоненты:

| Компонент | Описание |
|-----------|----------|
| **Retriever** | RAG поиск через ChromaDB |
| **Graph** | Граф связей между сущностями (NetworkX) |
| **Assembler** | Сборка контекста с бюджетом токенов |
| **Summarizer** | Сжатие истории бесед |
| **Archive** | Хранение сессий (сжатый JSON) |
| **Watcher** | Мониторинг изменений файлов |

## Установка

```bash
# Клонировать репозиторий
git clone https://github.com/Remchik64/pure-intellect.git
cd pure-intellect

# Создать виртуальное окружение
python3 -m venv venv
source venv/bin/activate

# Установить зависимости
pip install -e ".[dev]"

# Убедиться что Ollama запущена
ollama pull qwen2.5:7b  # или любая другая модель
```

## Запуск

```bash
# Запустить сервер Оркестратора
pure-intellect serve --port 8085

# Или через python
python -m pure_intellect serve --port 8085
```

## API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/chat` | Отправить запрос через Оркестратор |
| POST | `/index` | Проиндексировать проект |
| GET | `/status` | Статус системы |
| GET | `/graph` | Граф знаний |
| GET | `/docs` | Swagger UI |

### Пример запроса:

```bash
curl -X POST http://localhost:8085/chat \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Как работает аутентификация?",
    "mode": "explain",
    "model": "qwen2.5:7b"
  }'
```

### Пример индексации:

```bash
curl -X POST http://localhost:8085/index \
  -H "Content-Type: application/json" \
  -d '{"path": "/path/to/your/project"}'
```

## Конфигурация

Переменные окружения (префикс `PI_`):

| Переменная | По умолчанию | Описание |
|------------|-------------|----------|
| `PI_HOST` | `0.0.0.0` | Хост сервера |
| `PI_PORT` | `8085` | Порт сервера |
| `PI_OLLAMA_URL` | `http://localhost:11434` | URL Ollama |
| `PI_DEFAULT_MODEL` | `qwen2.5:7b` | Модель по умолчанию |
| `PI_MAX_CONTEXT_TOKENS` | `8192` | Макс. контекстное окно |
| `PI_STORAGE_DIR` | `./storage` | Директория хранения |
